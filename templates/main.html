Monogatari

Visual Novel Engine

Built to bring Visual Novels to the modern web and take them to the next level, making it easy for anyone to create and distribute Visual Novels in a simple way so that anyone can enjoy them on pretty much anywhere, create games with features that no one has ever imagined... it is time for Visual Novels to Evolve.

Every story should be told before they are lost forever.

A Simple Language

You don't need to be a developer to start building your VN, Monogatari features a Ren'py-like language designed to be incredibly simple and yet very powerful which will allow you to write your story as easy as i you where just writing it. Because of this, exporting Ren'py games to the web is very easy as well, you'll just need to add some commas and brackets here and there and your game will be ready for the web!

In this Multimedia World

We live in a world surrounded by media, images, sounds, videos and you can add any of them to your visual novel, add music to create a whole environment, sounds to make it even better and sure, you can add voices too so you'll characters will speak. If showing images and sounds is not enough, you can also show videos. Oh, those nice looking SVGs can be used too along with their incredible nice animations, as well as your favorite animated GIFs... All in a Responsive way, the perfect way to make your game playable in almost every device!

You are in Control

With Monogatari, you are in control of your visual novel like never before, change the font, backgrounds, colors, texts, positions, distribution etc. you can change, add and delete whathever you need in incredible ways. Think of your visual novel as a webpage, everything you've seen before on a webpage you can do it now in your visual novel.

Open Source

Monogatari is an open source project released under the MIT License which means you can use it, modify it, distribute it and pretty much everything so feel free to use it as you want!, there are no special conditions to use it altough all your players and I will really appreciate if you don't put any ads on your VN, adding ads is really easy since it's web based, but again, it's horrible for the users so please try not to do it.. The code is available in the GitHub Repository, feel free to take a look at it and contribute.

Built with Web Technologies

Monogatari is written using the latest HTML5, CSS3 and Javascript Features. The time we live in is amazing, there are lots of different devices, screens etc. that's why Monogatari is Responsive, it adapts to any screen so everyone will be able to enjoy your game, no matter what device they are on, besides since it's web based, they won't have to download anything, however it's not only limited to web, you'll also be able to create desktop and mobile apps!

Multi Language

Break any language barrier, since Monogatari is web based, you can write your novel in any language supported by the browsers, if you can write it with a keyboard you can use it, as simple as that. There's already built in functionality for you to translate your game into any language it's pretty simple and many people will really apreciate having finally visual novels in their language.

Extensible

Monogatari is built to be fully extensible, since you can add your own Javascript, CSS and HTML, you can pretty much do anything in your game, add animations, new screens etc. and if you need something fully integrated in the engine, you can modify it's code to add any features you'd like to it, if you take a look at the code you'll notice it's in fact very simple, it was made in a way any basic programming book will get you ready to understand almost everything and there's nothing in it that's not well documented in W3 Schools, you can also extend it with the whole bunch of APIs the web has to offer, so go ahead, you can mess around with it as much as you like to create amazing things.
